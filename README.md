# 🎲 Dice Roller App (Flutter)

A simple **Flutter app** that simulates rolling a dice. When you click the button below the dice image, the dice “rolls” and displays a **new random value**.

---

## 📱 Features

- Click the button to roll the dice  
- Dice image updates to show a **new random number (1-6)**  
- Smooth and intuitive UI for easy interaction  

---

## 🛠️ How It Works

1. The app displays a **dice image** at the center of the screen.  
2. A **button** is placed below the dice.  
3. When the button is pressed:  
   - A **new random number** between 1 and 6 is generated  
   - The dice image updates to reflect the new value
